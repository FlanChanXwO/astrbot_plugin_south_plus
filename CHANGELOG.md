# Changelog

## [1.0.2] - 2026-06-08

### Fixed
- **自动签到订阅三态计数修正**：首次新签成功计入 `✅`，已签到/请勿重复计入 `⏭️`，失败计入 `❌`
- **首次签到成功路径回归锁定**：补充调度链路测试，确保 `CheckinStatus.SUCCESS` 会在订阅推送中显示为成功

## [1.0.1] - 2026-06-07

### Changed
- **自动签到订阅推送格式优化**：区分全局订阅与会话订阅标题，并改为账号数、完成数、日签/周签三态统计格式
- **主动签到输出统一**：`/spcheckin` 改为统一状态格式，不再显示缓存或新签等内部细节
- **全体签到输出统一**：`/spallcheckin` 改为全体账号统计格式，并用 `请勿重复签到` 表示已签过项目

### Documentation
- **README 命令速览同步**：更新主动签到说明
- **项目概览同步**：更新主动签到与订阅推送格式说明

### Fixed
- **签到状态回归覆盖**：补充签到推送格式、主动签到格式、全体签到统计和失败/跳过状态的回归测试，锁定首次新签成功计入成功、已签到计入跳过的语义

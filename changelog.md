# Changelog

## 1.0.1 - 2026-06-07

### Changed

- 优化自动签到订阅推送格式，区分全局订阅与会话订阅标题。
- 自动签到订阅推送改为账号数、完成数、日签/周签三态统计格式，已签到跳过计入完成。
- 优化主动 `/spcheckin` 输出，改为统一状态格式，不再显示缓存或新签等内部细节。
- 优化管理员 `/spallcheckin` 输出，改为全体账号统计格式，并用 `请勿重复签到` 表示已签过项目。

### Documentation

- 更新 README 命令速览中的主动签到说明。
- 更新项目概览中的主动签到与订阅推送格式说明。

### Tests

- 补充签到推送格式、主动签到格式、全体签到统计和失败/跳过状态的回归测试。
